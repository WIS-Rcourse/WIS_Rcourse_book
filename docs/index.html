## Contributing to the book

Students are encouraged to contribute to the book giving feedback and proposing changes via GitHub (pull request, for experienced users). Mainly through the use of the disqus comments section at the bottom of each chapter. Note: Even if you are experienced with git and programming consider talking in the discussion section before investing a lof of time in the whole process. Let’s talk!
